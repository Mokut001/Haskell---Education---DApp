# ScholarsGate Multi-Page DApp

### Infrastructure
- **Backend**: Haskell Plutus Core (Majority Logic)
- **Frontend**: Next.js 13+ (App Router), Tailwind CSS
- **Deployment**: Vercel

### Setup
1. `npm install`
2. `./build.sh` (Requires Cabal/GHC for the backend portion)
3. `npm run dev` to start UI.
